# UPI-Transaction-Dashboard
UPI Transaction Dashboard built in Power BI showing monthly transaction volume, value, success rate, failure rate, and app-wise performance. Data cleaned using Power Query and KPIs created using DAX.
upi_data.csv contains UPI transaction-level data used for building the UPI Dashboard.
Fields include:
- Date
- Payment App
- Transaction Volume
- Transaction Value
- Success Rate
- Failure Rate
- Month
- User Count (if included)

UPI_Dashboard.pbix is the Power BI file containing the UPI Transaction Dashboard with success rate trends, app-wise comparison, and monthly transaction analysis.

upi_dashboard.png provides a snapshot of the UPI Transaction Dashboard built in Power BI.
